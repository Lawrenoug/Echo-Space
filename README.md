# Echo Space

`Echo Space` 是一个使用 `Godot 4.6.2 + C#` 开发的 2D 平台动作项目，核心方向是：

- 现实世界 / 灵魂世界实时切换
- 长横向多层地图探索
- 以《只狼》为参考方向的近战战斗

当前目标不是一次性把全部内容做完，而是先把程序底座、原型玩法和可迭代结构搭稳，再逐步补齐关卡、美术、音频和表现。

# 当前战斗方向

当前战斗设计方向已经明确为“血量 + 耐力 + 架势 + 处决”的组合：

- 玩家有血量和耐力
- 敌人有血量和架势
- 普通攻击会削减敌人血量，并附带少量架势伤害
- 防御可以减轻压力
- 精准弹反会更快累积敌人架势
- 当敌人架势打满后，会进入可处决窗口
- 当前原型中，对破绽敌人再次攻击会直接处决

# 当前默认按键

- `A` / `Left`：向左移动
- `D` / `Right`：向右移动
- `Space` / `W` / `Up`：跳跃
- `鼠标左键`：攻击 / 处决
- `鼠标右键`：防御 / 弹反
- `Tab`：切换世界
- `I`：打开 / 关闭背包界面
- `P`：打开 / 关闭加点界面
- `Esc`：关闭当前最上层系统界面

## 当前框架说明

- 项目启动入口已经切换为主菜单场景
- 主菜单当前提供：开始游戏、继续游戏、设置、退出
- 开始游戏已接入当前白盒关卡
- 继续游戏按钮已预留存档接口
- 设置面板中的显示 / 音频 / 按键 / 玩法按钮已预留后续接口
- 敌人框架已经拆成“通用战斗基类 + 具体敌人派生类”，后续新增敌人优先走继承复用
- 背包系统当前负责：物品定义、物品分类、槽位、堆叠、增删查、背包变化事件
- 加点系统当前负责：等级、未分配点数、属性分配 / 退点 / 重置、战斗修正值快照
- 二级系统界面当前采用统一的“面板栈”逻辑
- 新打开的界面会压到上层
- 下层界面不会被关闭
- 关闭当前界面后，下层界面会重新露出
- 后续新系统也应沿用这套方式
